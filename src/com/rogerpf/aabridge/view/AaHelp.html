Welcome to the aaBridge help file.

aaBridge is a simple Bridge playing app written by Roger Pfister see http://RogerPf.com

aaBridge is in tended for use by those new to the game of Bridge and by others who want practice at the rapid evaluaton of hands.. It always gives you good cards to play. :) aaBridge has been designed to be self documenting and all the features are intened to be discoverable just by using them, hence this help file is very short.

You can open a saved .lin deal file by Drag and Drop, just drop the file onto the aaBridge window.

The thick bars (right and bottom) are 'splitters', drag them to reveal the many dealing options.

Above all please remember to enjoy using the app, that after all is why we play.

And YES - the "automatic player" does CHEAT and peek at the hidden cards. Why? - So that you get to experience the best defense. Why would you want poor defense, holding such good cards, what would that teach you. :)

 

P.S. Any similarity between aaBridge and http://BridgeBaseOnline.com is intentional. :) It's a great site.

aaBridge is written in Java and so will run on Windows, MACs and Linux/Unix. It has been (will be) released as open source. The sources are available from (to be supplied).
